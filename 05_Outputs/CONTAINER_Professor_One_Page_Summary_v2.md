# CONTAINER: Open-Top Lunar Regolith Construction Machine

A proposed lunar construction concept currently being developed by Omar Galamli
Omar Galamli | Independent student

Status: Early-stage v2 concept; revised baseline and documentation in progress; physical prototype and test data not yet available.

## Plain-Language Concept

CONTAINER v2 is a proposed machine for early lunar construction. It is intended to land as a lighter system, collect local lunar regolith into a large open-top body, use that carried regolith as working mass, and help prepare landing-pad and berm infrastructure with a front excavation bucket and four long-reach robotic arms.

The core idea is simple: move light, work heavy.

## Operating Cycle

Land light -> Survey -> Excavate with front bucket -> Fill open container -> Manage center of mass -> Use side arms for handling/inspection/placement -> Build or support pad and berm operations -> Dump/place regolith -> Relocate lighter

## Current v2 Working Assumptions

These values are not measured performance data. They are working assumptions used to expose the highest-risk parts of the concept before prototype testing.

| Item | Proposed v2 assumption | Confidence |
|---|---:|---|
| Mission target | 14 m cargo landing pad + protective berm | Medium |
| Target lander class | 45-50 metric tons | Medium |
| Main body | 8 m x 6 m x 4 m open-top rectangular container | High |
| Excavation system | Front sliding-lift toothed bucket | Medium |
| Robotic system | 4 identical long-reach side-rail arms | Medium |
| Stabilization approach | Body geometry, carried regolith, controls; no anchors | Low |
| Sensor/camera reference | 48 distributed sensing locations | Medium |
| Power scenario | Solar + battery + possible FSP-class support | Low |

## What Changed From the Earlier Version

The earlier gantry-and-anchor architecture is retired. CONTAINER v2 no longer uses an elevated X-Y-Z gantry, helical anchors, stabilizer pads, outriggers, a closed top, or the old blade-auger-conveyor intake. Those files are preserved as historical context, not current design truth.

## Why Expert Feedback Is Needed

The concept is being refined before prototype work. The most uncertain v2 assumptions are no-anchor stability, bucket excavation/lift loads, side-arm mass and stiffness, dust durability, underfloor thermal management, mobility/ground pressure, sintering or pad-surface construction method, and battery/shadow survival.

Feedback requested: What v2 assumption seems most likely to fail first, and what would be the smallest useful test to measure it?

More information is included in the v2 baseline and one-page brief.
