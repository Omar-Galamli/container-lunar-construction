# CONTAINER Digital Prototype

This folder contains the first browser-based visual prototype for CONTAINER.

Open `index.html` in a browser or serve this folder locally.

## What v1 Shows

- Lunar worksite scene with pad, berm, CONTAINER platform, anchors, gantry, intake, and tile field.
- Operating phase controls: deploy, fill, anchor, construct, berm, relocate.
- Interactive assumptions for fill rate, ballast, tile energy, battery capacity, FSP, and survival load.
- Live outputs for fill time, stability margin, energy balance, battery survival, and rough schedule.

## Status

This is a concept visualization and model dashboard. It is not a CAD model, physics simulator, or validated engineering tool.
