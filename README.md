# CONTAINER: Lunar Surface Construction Concept

CONTAINER is an early-stage lunar construction concept built around the idea of moving light and working heavy: land a comparatively light machine, collect local regolith into the machine body, use that carried mass during work, and place or dump the regolith into useful site infrastructure.

The current public baseline is CONTAINER v2. It is an open-top 8 m x 6 m x 4 m regolith container with a front sliding-lift bucket, four side-rail robotic arms, a rear-access underfloor equipment deck, distributed sensors/cameras, and shielded fixed body panels. The earlier gantry, helical anchors, stabilizers/outriggers, closed top, and blade-auger-conveyor intake are retired from the current baseline.

## Status

This is a concept-stage engineering repository. Assumptions, model outputs, geometry, and design targets are not proven performance claims. The v2 package is meant to make the idea reviewable, testable, and easier to critique before detailed mechanical design, FEA, actuator sizing, thermal analysis, materials selection, and prototype testing.

## Historical Context

Files without a `_v2` suffix generally describe the older v1 architecture. They are kept as historical design evidence and should not be treated as the current CONTAINER baseline unless a document explicitly says so.

## Public Scope

This public package intentionally excludes private outreach records, individual contact lists, raw downloaded source libraries, temporary render files, virtual environments, and large generated research archives.
