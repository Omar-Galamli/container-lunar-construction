# CONTAINER: Lunar Surface Construction Concept

CONTAINER is an early-stage lunar surface construction concept for a mobile construction station that uses locally collected regolith as ballast and working material. The project explores how a lightweight deployed system could become mechanically heavier after landing, then support excavation, anchoring, precision tooling, and surface preparation work.

This repository is the public-facing technical package. It includes the current baseline, assumptions, requirements, early model, browser prototype, and selected written outputs.

## Status

This is a concept-stage engineering repository. Assumptions, model outputs, and design targets are not proven performance claims. The current goal is to make the technical idea reviewable, testable, and easier to critique.

## Public Scope

This public package intentionally excludes private outreach records, individual contact lists, CAD/KCL snapshots, orthographic drawings, raw downloaded source libraries, temporary render files, virtual environments, and large generated research archives.
